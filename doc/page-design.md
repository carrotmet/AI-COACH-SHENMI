本文档详细描述了深觅 AI Coach 系统的四个关键页面的设计规范和实现指南。

### 1.3 设计细节

| 元素 | 样式 |

| 元素 | 样式 |

- **桌面 (≥ 1024px)**：3列功能卡片，水平按钮

### 1.5 交互效果

- 功能卡片悬停：上移 4px，阴影加深
- 按钮悬停：背景加深，轻微上移
- 页面滚动：导航栏添加阴影

## 6. 动画规范

### 6.1 过渡时间

| 类型 | 时间 | 用途 |
|------|------|------|
| Fast | 150ms | 颜色、透明度变化 |
| Normal | 250ms | 位置、大小变化 |
| Slow | 350ms | 复杂动画 |

### 6.2 缓动函数

| 类型 | 函数 | 用途 |
|------|------|------|
| Ease Out | `cubic-bezier(0, 0, 0.2, 1)` | 进入动画 |
| Ease In Out | `cubic-bezier(0.4, 0, 0.2, 1)` | 通用过渡 |
| Spring | `cubic-bezier(0.175, 0.885, 0.32, 1.275)` | 弹性效果 |

### 6.3 页面过渡

- 页面切换：淡入淡出，250ms
- 模态框：从底部滑入，350ms
- Toast：从顶部滑入，250ms

## 7. 实现建议

### 7.1 技术栈

- **CSS 框架**：原生 CSS + CSS 变量
- **动画**：CSS Transitions + CSS Animations
- **图标**：SVG 图标或图标字体
- **响应式**：CSS Media Queries

### 7.2 性能优化

1. 使用 `will-change` 优化动画性能
2. 图片懒加载
3. 虚拟滚动处理长列表
4. 减少重绘和重排

### 7.3 无障碍

1. 所有交互元素支持键盘操作
2. 图片添加 alt 属性
3. 颜色对比度符合 WCAG 标准
4. 支持屏幕阅读器

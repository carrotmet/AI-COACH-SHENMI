## 版本号规范

本项目遵循 [语义化版本 2.0.0](https://semver.org/lang/zh-CN/) 规范：

```
版本格式：主版本号.次版本号.修订号（MAJOR.MINOR.PATCH）

- MAJOR：不兼容的 API 修改
- MINOR：向下兼容的功能性新增
- PATCH：向下兼容的问题修正
```

**版本标签说明：**

| 标签 | 说明 |
|------|------|
| 🚀 | 重大更新 |
| ✨ | 新增功能 |
| 🎨 | 功能优化 |
| 🐛 | Bug修复 |
| 📝 | 文档更新 |
| 🔒 | 安全更新 |
| ⚡ | 性能优化 |
| ♻️ | 代码重构 |

## 快速参考

### 如何添加新版本

1. 复制 `[Unreleased]` 部分的内容
2. 创建新版本标题：`## [版本号] - YYYY-MM-DD`
3. 在版本历史记录表中添加新行
4. 更新 `[Unreleased]` 部分为空白模板

### 如何标记变更类型

| 变更类型 | 前缀 | 示例 |
|----------|------|------|
| 新增功能 | ✨ | ✨ 新增用户注册功能 |

| Bug修复 | 🐛 | 🐛 修复登录失败问题 |
